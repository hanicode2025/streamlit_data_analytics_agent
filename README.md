# AI-Powered Data Analyst Agent

**Overview:**  
Streamlit-based interactive tool for data analysis and visualization. Users can explore datasets, view tables, generate plots, and parse multiple document formats.

**Technologies Used:**  
- Python 3.x  
- Pandas, NumPy  
- Streamlit  
- Matplotlib, Seaborn  

**Features:**  
- Upload datasets (CSV, Excel, etc.)  
- Interactive visualizations (line plots, bar charts, heatmaps)  
- Table view for raw data  
- AI-assisted document parsing and data summarization  

**Key Skills Demonstrated:**  
- Data analysis and visualization  
- Streamlit web app development  
- Multi-format data handling (CSV, Excel, text)  
- Python scripting and automation  

**Project Structure:**  
AI-powered-Data-Analyst-Agent/
├── app.py # Streamlit main app
├── data_loader.py # Handles dataset import & preprocessing
├── visual.py # Visualization functions
├── llama_interface.py # AI / LLM interaction module
├── ask_llama.py # LLM query handler
├── main.py # Entry point
├── requirements.txt # Dependencies
├── README.md # Project documentation
└── example_outputs/ # Optional folder with sample visualizations
